## Debugging Tips
Steps:

1. **Compile** all files with the `-g` flag.

2. Start **debugging**:
```
lldb NAME_OF_YOUR_PROGRAM
```

To set a **breakpoint** in line 3:
```
b 3
```

To see the **value of a variable**:
```
p var_name
```
